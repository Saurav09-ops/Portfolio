CET138 · Full Stack Development

Understanding the building blocks of
full stack development.

This portfolio documents my learning journey throughout the Full Stack Development module. It demonstrates my understanding of core web technologies including HTML, CSS, Bootstrap, and JavaScript, before bringing these skills together in the development of a basic full stack application. Through these demonstrations, I explore how front-end, back-end, and database technologies combine to create complete and functional web applications.
